⚔️ DARK KNIGHT 2D – Animated Character & VFX

Thank you for downloading Dark Knight 2D.

This asset provides a fully animated 2D character with integrated VFX, designed as a ready-to-use Unity prefab for rapid prototyping and gameplay integration.

⚔️ Features
Fully animated 2D character (Idle, Run, Walk, Attack, Death, etc.)
Integrated VFX system (beam, dissolve, trail effects)
Clean Unity prefab hierarchy
Custom scripts included for basic character control
Audio support included (44.1 kHz)
Designed for quick integration into any project

🧠 Usage

Drag and drop the prefab into your scene.

No additional setup required beyond standard Unity configuration.
